# Orbit Startpage

A minimal, highly customizable terminal-inspired startpage and Chrome extension.

## Features
- Command palette interface (`:help` for shortcuts)
- Live search suggestions (Google / DuckDuckGo fallback)
- Custom bookmarks with auto-detecting Simple Icons
- Built-in UI for customizing colors, background, and ASCII animations
- Settings sync locally across your browser

## How to Install (Chrome Extension)

To use this as your default New Tab page in Chrome, Brave, or Edge:

1. Download or clone this repository to a folder on your computer.
2. Open your browser and go to `chrome://extensions/`
3. Turn on **Developer mode** (usually a toggle in the top right corner).
4. Click the **Load unpacked** button in the top left.
5. Select the folder containing these files.

That's it! Open a new tab to see your new startpage. Use `:config` to customize it.
